# inline::pypdf

## Description

Simple PDF text extraction using PyPDF library. Default processor for backward compatibility.

## Configuration

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `timeout_seconds` | `<class 'int'>` | No | 30 | Processing timeout in seconds |
| `max_file_size_mb` | `<class 'int'>` | No | 50 | Maximum file size in MB |

## Sample Configuration

```yaml
timeout_seconds: 30
max_file_size_mb: 50
```
